# HTML 双向编辑器

<p align="left">
  <a href="./README.md">English</a> | 简体中文
</p>

这是一个轻量级的 HTML 双向编辑器，采用纯前端实现，支持实时预览与双向内容同步。
## 🎯 核心使用场景：AI简历微调、PDF导出
> 现在用 Codex、Claude Code 等 AI 工具生成 HTML 格式的简历非常方便，样式美观且结构清晰。但在实际投递前，我们往往需要针对不同岗位进行细节微调（改个错别字、调个间距、换个排版等等）。

- **双向实时同步**：左侧编辑 HTML 源码，右侧实时预览渲染效果；同时支持直接在右侧预览区修改内容，变更会自动同步回左侧源码。
- **零成本运行**：极致轻量，即开即用，完全基于前端技术开发，无需任何后端服务，也无需复杂的构建工具。
- **轻松转PDF**：微调结束后，直接在浏览器中按下 `Ctrl + P` (或 `Cmd + P`)，打印另存为 PDF。

## 📦 快速开始

由于这是一个纯前端项目，你不需要安装任何依赖即可直接运行：

1. 克隆项目到本地：
```bash
   git clone git@github.com:willcz23/html-online-editor.git
   cd html-online-editor
```
2. 双击 index.html 即可在浏览器中打开。
