# HTML 双向编辑器

<p align="left">
  <a href="./README.md">English</a> | 简体中文

这是一个轻量级的 HTML 双向编辑器，采用纯前端实现，支持实时预览与双向内容同步。

## 🌟 项目亮点

- **双向实时同步**：左侧编辑 HTML 源码，右侧实时预览渲染效果；同时支持直接在右侧预览区修改内容，变更会自动同步回左侧源码。
- **纯前端实现**：完全基于前端技术开发，无需任何后端服务，也无需复杂的构建工具（如 Webpack/Vite）。
- **零依赖**：极致轻量，即开即用，无任何第三方库负担。
